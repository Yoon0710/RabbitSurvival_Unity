# RabbitSurvival

Author - Yoon0710

This is a shotting game motivated by Vampire Survivors.

# Requirement

- Unity 2021.3.5f1

## Key input

| Key   | Function               |
| ----- | ---------------------- |
| LEFT  |  Move left              |
| RIGHT |  Move right             |
| DOWN  |  Move down              |
| Z     |  Cast attack            |
| X     |  Cast Skill             |
