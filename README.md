# RabbitSurvival

Author - Nero0710

This is a shotting game motivated by Vampire Surviviors.

## Key input

| Key   | Function               |
| ----- | ---------------------- |
| LEFT  |  Move left              |
| RIGHT |  Move right             |
| DOWN  |  Move down              |
| Z     |  Cast attack            |
| X     |  Cast Skill             |
